Assignment 1

The goal of this assignment is to perform calculations using Weight objects which are instances of Weight class.

This assignment consists of two classes: Weight.java and Project.java

Weight.java

The Weight.java class has three private variables, two private methods and four public methods

Private variables for Weight.java

A private constant variable called OUNCES_IN_A_POUND that defines the number of ounces in a pound (16).

A private variable called pounds with a data type of integer.

A private variable called pounds with a data type of integer.

A private variable called ounces with a data type of double.

Methods for Weight.java

A public parameterized constructor, which initializes the private variables: public Weight (int pounds, double ounces).

A private instance method called toOunces with a return type of double. This method has no parameters and should return the total number of ounces.

A private instance method called normalize with a return type of void. This method has no parameters and ensures the he number of ounces is less than the number of ounces in a pound.

A public instance method called lessThan with a return type of boolean. This method will accept a Weight object as a parameter and determine if the object is greater or less than the initialized values :public boolean lessThan (Weight weight)

A public instance method called addTo with a return type of void. This method will accept a Weight object as a parameter and add the object’s weight values to the initialized values: public void addTo (Weight weight)

A public instance method called toString with a return type of String. This method has no parameters and should have the following format: x pounds and y ounces where x is the number of pounds and y the number of ounces. Ounces will be displayed with two decimal places.

Project1.java

Project1 Class has three private methods and one public method.

Methods for Project1.java

A private class method named findMinimum with a return type of Weight. This method will accept three Weight objects as parameters and compare each Weight object’s weight values to find the minimum. The Weight object with the minimum weight value will be returned and then printed using toString in the following format: The minimum weight is x pounds and y ounces where x is the number of pounds and y the number of ounces. Ounces will be displayed with two decimal places.

A private class method named findMaximum with a return type of Weight. This method will accept three Weight objects as parameters and compare each Weight object’s weight values to find the maximum. The Weight object with the maximum weight value will be returned and then printed using toString in the following format: The maximum weight is x pounds and y ounces where x is the number of pounds and y the number of ounces. Ounces will be displayed with two decimal places.

A private class method named findAverage with a return type of Weight. This method will accept three Weight objects as parameters and calculate the average weight value. A new Weight object with the average weight values will be returned and then printed using toString in the following format: The average weight is x pounds and y ounces where x is the number of pounds and y the number of ounces. Ounces will be displayed with two decimal places.

A public method named main with a return type of void. This method should exercise the correct functionality of findMinimum, findMaximum and findAverage by creating three Weight objects using the hardcoded values below: Weight weight1 = new Weight(11, 3); Weight weight2 = new Weight(7, 20);

Assignment 1 results when the user inputs (11,3), (8,4), and (14,6) for weight 1, weight 2, weight 3 respectively
Assignment 1 results when the user inputs (26,5), (11,9), and (11,6) for weight 1, weight 2, weight 3 respectively
Assignment 1 results when the user inputs (15,4), (8,8), and (14,6) for weight 1, weight 2, weight 3 respectively
