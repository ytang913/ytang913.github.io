About

This project is a Login Registration Application built using React.JS, Express, MongoDB,Bootstrap CSS, and axios

Users are tasked to enter in four fields: Full Name, Username, Email, and Password

Documentation

Backend

Express and MongoDB was used for this portion.

server.js is created for the backend server and also connected to Mongoose

routes.js is created to route the requests that come to the server

LoginModels.js is created for the schema and Mongoose is housed here and within it there are five fields Full Name, Username, Email, password, and Date.

The Four fields: Full Name, Username, Email, and password is what the user will enter in, while the Date field will allow me to know which month,day, and year they entered in their information.

MongoDB Cloud Atlas is used to house the data received from Post requests and it allows me to see the requests.

Frontend

React.JS was used for this portion.

When a user enters in their information a POST request is sent and router.post will handle it

The Post request will also create a new schema

Created a cluster on mongo atlas powered by AWS

super class and constructor to hold on the states for the login registration

we are using the setState function to change the events

When the user puts their information in the empty form, the onSubmit event will be triggered then the methods will be triggered. When the user clicks on submit, the onSubmit event will be triggered and it will gather the values that the user entered which are stored in the states. They will save it in registered. We are then gonna send registered into the backend. which is done using axios

Whatever the user is typing will be saved in states
