DannyNoteTaker Web App

For this project, my goal is to build a Note Taking App for myself using HTML, CSS, and JavaScript.

In this project, users are able to add and delete their notes on the Web app itself. There is an "Add" button to create a note, a "trash can" button to delete the note, and an "pencil" button to save their note.

The web app was deployed on Microsoft Azure Cloud
