About

There is the lack of accessible information on the communities of Prince Georges County for many of its inhabitants. Specifically, the number of crimes and types of crimes

This project is a web-based application that is designed to solve this problem by allowing users can see navigate accurate and organized PG County crime via the search and data functionality.

The users will be able to see information related to security and safety concerns within different areas of Prince Georgeâ€™s county. More specifically, users would be able to search an address within our search feature. From there, they would be able to see the corresponding information based on the address they entered.

This web-based application is split up in two sections Home and Data

Documenation

This web-based application was created using HTML, CSS, JavaScript, APIs, JSON file, Leaflet, SQLite, Express, and Node.Js.

The web based application was deployed on: Microsoft Azure Cloud and Heroku
