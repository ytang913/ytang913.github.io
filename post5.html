About

The purpose of making this full stack application was to create a To Do List using both front end and back end technologies.

There will be five sections present in this full stack application: Add New Task, Get Task Id, Enter the Task ID to update then press 'Get Task' button, Delete task by task Id, and Add new Person

Add New Task

In this section, where the user can add a new task in the "Add Task name" section. Then the user and choose which group the task belongs in, the name of the person that the task applies to, and finally the priority of the task.

Documenation

This project is done in Visual Studio Code using HTML, CSS, JavaScript, Express, and SQLite.

HTML and CSS is used to design the overall application.

JavaScript, Express, and SQLite is used for the backend side of the application.

This application can run locally just type in npm run start-watch in the terminal.

Once npm run start-watch is entered in the terminal the first time, the application will start up and the script run that will create a SQLite database called 'db.toDoList'.

This database should be able to be viewed on DB browser for SQLite.

Once npm run start-watch ran succesfully, then the application can be accessed locally at http://localhost:4000/
